Projeto simples de lista de tarefas desenvolvido com **HTML, CSS e JavaScript puro**


🔗**Acesse o site clicando no link a seguir:** https://renanromi.github.io/To-do-List/


**Como funciona:**


O To-Do list é um projeto de interface e código simples, sua função é adicionar e remover tarefas do dia a dia, perfeito para o primeiro projeto e aprender na prática como desenvolver utilizando HTML, CSS e JavaScript.


**Tecnologias utilizadas:**

- HTML 5
- CSS3
- JavaScript (DOM, manipulação de eventos)

**O que aprendi com esse projeto?**

- Manipulações DOM
- Eventos de clique (`onclick`)
- Fronted e estilização com CSS (transições, hover, flexbox)


**Próximos passos**

- Salvar tarefas com localStorage
- Integrar com um banco de dados (Firebase)
- Adicionar opção de marcar tarefa como concluida
